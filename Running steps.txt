
****************  Running Steps  ****************


1. First Download & Install XAMPP/MAMP/WAMP server.

2. Text Editor or IDE: any IDE or Text Editor you wish to use.

3. Download the zip file(code)

4. Extract the file and copy folder in which zip file extracted (for eg. BANK ) and

5. Paste inside root directory where you installed xammp/mamp/wamp local server
    paste: (for eg. xampp/htdocs/   )

6. Open PHPMyAdmin (http://localhost/phpMyAdmin)

7. Create a database with name "banking_system"

8. open "connection.php" file and replace the host, username, password, port variales with your system configuration.

8. Import customers.sql and transactions.sql files
    (given inside the zip package in "MySQL Database" folder)

9. Run the script browser by typing "http://localhost/bank"
